Instructions on importing the examples into an Eclipse workspace.

 Alternatively you can just extract it to any directory and view the
contents.  Importing, allows you to easily run the
=E2=80=8B =E2=80=8B
examples yourself in Eclipse.

To import
1.  In an eclipse workspace, click on File -> Import
2.  Expand the General folder and select  "Existing Projects Into
Workspace", then click Next.
3.  Select "Select an archive file" then click Browse
4.  Find, and select the file you downloaded containing the examples, then
click Open.
5.  Click Finish
